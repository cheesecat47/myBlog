## 🏃‍♂️ Run Locally

### Client 실행

1. 원격 저장소를 복제합니다.

```bash
  git clone https://github.com/rosielsh/my-blog
```

2. 프로젝트 폴더로 이동합니다.

```bash
  cd my-blog
  cd frontend
```

3. 필요한 node modules를 설치합니다.

```bash
  npm install
```

4. 서버를 실행합니다.

```bash
  npm run start
```
