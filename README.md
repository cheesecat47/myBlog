# myBlog

> 블로그 서비스를 만들어보는 프로젝트입니다.

## Tech Stack

- FE: Vue.js
- API: Spring Boot, MyBatis, MySQL

## Run Locally

- 프로젝트 클론 후 디렉토리로 이동

    ```bash
      git clone https://github.com/cheesecat47/myBlog.git
      cd myBlog
    ```

### FE Dev server

- 실행 전 아래 요구사항이 설치되어 있는지 확인이 필요합니다.
- Node v18

1. `myBlog-frontend` 디렉토리로 이동

  ```bash
    cd myBlog-frontend/
  ```

1. 실행

  ```bash
      npm install
      npm run dev
  ```
